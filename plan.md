# CalBot — План разработки

> Метод: вайбкодинг — код пишет Claude, человек проверяет результат в Telegram после каждого шага.
> Каждый этап даёт видимый результат. Следующий этап начинается только после проверки предыдущего.

Минимум, который можно показать реальным пользователям.

### Фундамент
- [x] Создана структура проекта (`handlers/`, `services/`, `database/`, `scheduler/`)
- [x] Настроены зависимости (`requirements.txt`)

### Онбординг
- [x] Приветственный экран с кнопкой "Поехали!"
- [x] 5 вопросов по одному: рост → вес → возраст → цель → активность
- [x] Прогресс-бар на каждом шаге ("■■□□□ Шаг 2 из 5")
- [x] Расчёт нормы калорий по формуле Миффлина–Сан Жеора
- [x] Повторный `/start` не запускает онбординг заново

### Распознавание еды по фото
- [x] Мгновенный ответ "Анализирую..." после получения фото
- [x] Claude Vision распознаёт блюдо и возвращает КБЖУ
- [x] Показывает: блюдо, калории, белки, жиры, углеводы, совет
- [x] Кнопки: ✅ Добавить / ✏️ Изменить / ❌ Не добавлять
- [x] Сценарий ошибки: если еда не распознана — кнопка ручного ввода

### Дневник питания
- [x] Записи сохраняются в SQLite
- [x] После добавления показывается счётчик дня (съедено / норма)
- [x] Команда `/diary` — весь список за сегодня
- [x] Кнопка "Отменить последнюю запись"

### Автоматика
- [x] Вечерний отчёт в 21:00 МСК (APScheduler)
- [x] AI-коуч пишет мотивацию при сериях 3 / 7 / 14 / 30 дней

### Деплой
- [x] Код на GitHub (приватный репозиторий)
- [x] Бот запущен на Render (бесплатный Web Service)
- [x] UptimeRobot пингует каждые 5 минут — бот не засыпает
- [x] Бот работает 24/7 без компьютера
